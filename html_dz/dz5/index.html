Команди якими я користуюсь Властивість
git status Сама головна комнада яка показує стаy ваших файлів
git init Створює git-репозиторій у папці
git add "файл" додає файл до підготовки
git add . Додає всі файли до підготовки
git commit -m "опис зиіни" Зберігає зміни в історії Git
git branch Показує гілки
git push Відправляж коміт на GitHub
